ShiftLoL

ShiftWindow for League of Legends players

Download ShiftLoL v1.3.0 beta Download ShiftLoL v1.2.1 View Project on Github

Features

  • Supports "Windowless" and "Bordered" window modes!
  • Moves the LoL window with a single click/key!
  • Plays a sound when game starts!
  • and nothing else!

Known Problems (v1.3.0 beta)

  • "Windowed" mode is buggy on Windows 8, because when you "shift" the LoL window, taskbar gets in front of the LoL window. A solution for this which I haven't tested.
