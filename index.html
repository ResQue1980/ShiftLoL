ShiftLoL

ShiftWindow for League of Legends players

Download ShiftLoL v1.2.1 View Project on Github

Features

  • Supports "Windowless" and "Bordered" window modes!
  • Shifts the LoL window with a single click/key!
  • Plays a sound when game starts!
  • and nothing else!

Why should I use this?

You will be supporting an open source project!
